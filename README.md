## توضیح کلی پروژه
هدف این پروژه طراحی و پیاده‌سازی یک پلتفرم هوشمند آموزش زبان است که با استفاده از رویکرد یادگیری تطبیقی مسیر آموزشی هر کاربر را بر اساس سطح مهارت، سرعت یادگیری و میزان پیشرفت او شخصی‌سازی می‌کند.
مسئله‌ی اصلی این پروژه، فراهم‌ کردن بستری برای آموزش مؤثر و پویا است تا به جای ارائه‌ی محتوای یکسان برای همه، سیستم بتواند متناسب با نیاز هر زبان‌آموز، تمرین‌ها و درس‌های مناسب را پیشنهاد دهد. این موضوع باعث افزایش بهره‌وری در فرآیند یادگیری و انگیزه‌ی بیشتر کاربران می‌شود.
کاربرد این پلتفرم در مراکز آموزشی، دوره‌های آنلاین، و آموزش‌های فردی است. کاربران می‌توانند درس‌ها و تمرین‌های تعاملی انجام دهند، سطح زبانی خود را مشاهده کنند، و در صورت نیاز از طریق گفت‌وگوی متنی یا صوتی با مربی در ارتباط باشند.


## اهداف پروژه
هدف 1: شخصی سازی مسیر یادگیری زبان
هدف 2: افزایش تعامل کاربر با سیستم آموزشی
هدف 3: بهبود ارزیابی سطح زبان کاربران
هدف 4: ایجاد ارتباط مستقیم میان زبان‌آموز و مربی
هدف 5: توسعه یک پلتفرم قابل گسترش و کاربر‌پسند


## فناوری‌ها و ابزار‌های پیشنهادی
زبان برنامه نویسی: Python
برای پیاده سازی منطق یادگیری تطبیقی و توسعه بخش های تحلیلی
فریم ورک: Django,React
از Django برای پیاده سازی سریع و ایمن بخش سرور و مدیریت کاربران، درس‌ها و تمرین ها ( سازگار با Python )
از React برای ساخت رابط کاربری پویا و جذاب که تجربه کاربری روانی ارائه می‌دهد ( سازگار با Javascript )
ترکیب این دو، ترکیبی سریع،امن و حرفه ای برای وب اپلیکیشن
پایگاه داده: PostgreSQL
 برای داده های پیچده و پشتیبانی از پردازش های تحلیلی ( سازگاری بالا با Django و داده های تحلیلی )
