## نیازمندی‌های کاربر ( User Requirements )
1-ثبت نام و ورود کاربران با ستفاده از ایمیل یا حساب کاربری گوگل

2-مشاهده‌ی درس ها و تمرین ها باساس سطح فعلی کاربر

3-افزایش یا کاهش سطح زبان براساس نتایج آزمون ها و تمرین ها

4-تعامل متنی یا صوتیبا مربی جهت رفع اشکال یا تمرین مکالمه

5-مشاهده گزارش پیشرفت شامل امتیازات، سطح و درصد پیشرفت

6-دسترسی به محتوای آموزشی از طریق تلفن همراه و رایانه

## نیازمندی‌های سیستمی ( System Requirements )
1-سیستم مدیریت کاربران یزای ثبت، ورود و سطح‌بندی کاربران

2-ماژول مدیریت درس ها و تمرین ها برای بارگذاری، ویرایش و نمایش محتوای آموزشی

3-سیستم ارزیابی و تعیین سطح خودکار براساس عملکرد کاربر

4-پشتیبانی از گفت و گو های متنی و صوتی بین کاربر و مربی

5-ذخیره سازی داه ها در پایگاه داده

6-رابط کاربری Responsive جهت سازگاری با موبایل و دسکتاپ

## نیازمندی‌های غیرکارکردی ( Non-Functional Requirments )
1-امنیت

2-مقیاس‎پذیری

3-رابط کاربری کاربر پسند
